input_parameters.rf_parameters module¶

Module gathering and processing all the RF parameters to be given to the other modules.

Authors:Alexandre Lasheen, Danilo Quartullo
class input_parameters.rf_parameters.RFSectionParameters(n_turns, n_rf_systems, section_length, harmonic_number_list, voltage_program_list, phi_offset_list, momentum_program)¶

Object gathering all the RF parameters for one section (see section definition in longitudinal_tracker.Ring_and_RF_section), and pre-processing them in order to be used in the longitudinal_tracker.py module. It can be added to another RF_section_parameters object by the Sum_RF_section_parameters object in order to concatenate all the parameters for one full ring.

harmonic_number_list = None¶
Harmonic number list : \quad h_{j,n}
The length of the list should be equal to n_rf_systems.
index_section = None¶

Index of the section (incremented by one whenever a RFSectionParameters is created.

momentum_program = None¶
Momentum program in [eV/c] : \quad p_{j,n}
The length of the momentum program should be n_turns + 1, check longitudinal_tracker.py for more precisions.
Inputing a single value will assume a constant value for all the simulation.
n_rf_systems = None¶
Number of RF systems in the section : \quad n_{RF}
Counter for RF is: j
n_turns = None¶
Number of turns for the simulation
Counter for turns is: n
p_increment = None¶

Momentum increment (acceleration/deceleration) between two turns, for one section in [eV/c] : \quad \Delta p_{n\rightarrow n+1}

phi_offset_list = None¶
Phase offset list in [rad] : \quad \phi_{j,n}
The length of the list should be equal to n_rf_systems.
section_counter = -1¶
section_length = None¶

Length of the section in [m] : \quad L_i

voltage_program_list = None¶
Voltage program list in [V] : \quad V_{j,n}
The length of the list should be equal to n_rf_systems.
class input_parameters.rf_parameters.SumRFSectionParameters(RFSectionParameters_list)¶

Method to add RF_section_parameters objects together in order to gather the complete information for a longitudinal_tracker.Full_Ring_and_RF object

RFSectionParameters_list = None¶

List of RF_section_parameters objects to concatenate

momentum_program_matrix = None¶
Momentum program matrix in [eV/c] : \quad p_{i,n}
The lines i of this matrix corresponds to the momentum program for one section.
The columns n correspond to one turn of the simulation.
section_length_sum = None¶

Total length of the sections in [m]

total_n_sections = None¶
The total number of sections concatenated
Counter for section is: i
input_parameters.rf_parameters.input_check(input_value, expected_length)¶
Function to check the length of the input
The input can be a float, int, np.ndarray and list
If len(input_value) == 1, transform it to a constant array
If len(input_value) != expected_length and != 1, raise an error
